Lavoro 6: Secret

Si realizzi un software web che importi un file di testo contenente il brano da criptare e il dizionario personalizzato (eventualmente anche di un altra lingua). L'app sarà composta da due pagine: la prima che importi attraverso l'oggetto fileReader il file di partenza. La seconda pagina html, attivabile attraverso un tasto oppure raggiungibile via url, dovrà stampare il brano criptato e permettere, attraverso una "input type=text", la chiave per decriptare il messaggio e stamparlo. Non essendo possibile modificare il file di partenza, si utilizzi il localStorage per "storare" la stringa criptata. In allegato l'immagine delle pagine
